# WordPress Dog Toy Landing Page

This folder contains a sample WordPress page template that you can use to create a landing page for a dog toy.

## Installation

1. Copy `page-dog-toy-landing.php` into your active theme directory (usually `wp-content/themes/your-theme/`).
2. Upload an image named `dog-toy-hero.jpg` to your theme's `images` folder or adjust the path in the template.
3. In the WordPress admin panel, create a new page and select **Dog Toy Landing Page** under **Page Attributes > Template**.
4. Publish the page.

The template displays a hero image, a clear value proposition with a title and subtitle, calls-to-action, customer testimonials via the `[testimonials]` shortcode, a FAQ section, and a footer with a privacy policy link.
